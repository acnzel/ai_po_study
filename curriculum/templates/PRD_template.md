## 3. 사용자 시나리오

### As-Is (현재 — 이 기능 없을 때)

```
1. 사용자가 마음에 드는 상품을 찜한다
2. 해당 상품이 품절된다
3. 사용자는 이 사실을 모른 채 나중에 상품 페이지를 직접 방문한다
4. 품절을 확인하고 실망한 후 이탈한다
```

### To-Be (이 기능이 있을 때)

```
1. 사용자가 마음에 드는 상품을 찜한다
2. 해당 상품이 품절된다
3. 상품이 재입고되면 앱 푸시 알림을 받는다
4. 알림 탭하면 상품 페이지로 이동한다
5. 바로 구매할 수 있다
```
