# 📚 초보 PO를 위한 Claude 활용 커리큘럼
## "AI와 함께 PRD·기획 문서 쓰는 법"

> **대상**: Claude를 처음 쓰거나, 쓰고 있지만 제대로 활용하지 못하는 PO  
> **목표**: 기획 문서 작성 시간 50% 이상 단축, 문서 품질 향상  
> **기반**: Lablup 신정규 대표 EP.86 "진짜 내 일을 해결하는 Agentic Workflow"

---

## 이 커리큘럼을 마치면 할 수 있는 것

- ✅ Claude와 단계적으로 대화하며 PRD 초안을 40분 안에 완성할 수 있다
- ✅ 나만의 Soul Document를 만들어서 매번 같은 설명을 반복하지 않을 수 있다
- ✅ 자주 쓰는 업무(스프린트 리뷰, 기능 분석 등)를 명령어 하나로 자동화할 수 있다
- ✅ 에이전트에게 검토를 맡겨서 내가 놓친 엣지케이스를 발견할 수 있다
- ✅ 나만의 PO-Claude 업무 Workflow를 설계할 수 있다

## 학습 방법 가이드

### 🟢 완전 처음이라면 (1-2주 코스)
```
1주차: 1강 → 2강 → 3강 (기초 익히기)
2주차: 4강 → 5강 → 6강 (실무 적용)
이후: 7강 → 8강 (심화)
```

### 🟡 Claude 써봤는데 잘 모르겠다면 (1주 코스)
```
바로 3강(Soul Document)부터 시작
→ 4강(PRD) → 6강(자동화) → 8강(종합)
```

### 🔴 급하게 당장 써야 한다면 (당일 코스)
```
2강(기초 프롬프팅) 빠르게 읽기
→ templates/ 폴더에서 PRD 템플릿 가져가기
→ 부록 프롬프트 복사해서 사용
```

## 이 커리큘럼의 핵심 철학

> *"내 걸 만들게 되면 그 가속이 시작됩니다."*  
> — 신정규 (Lablup 대표)

남이 만든 프롬프트를 복사하는 것도 좋지만, **나만의 시스템을 만드는 순간** 진짜 가속이 시작됩니다.

이 커리큘럼은 단순히 "Claude 잘 쓰는 법"이 아니라, **나만의 AI 협업 시스템**을 만드는 과정입니다.
