# 📚 초보 PO를 위한 Claude 활용 커리큘럼

### AI와 함께 PRD·기획 문서 쓰는 법

> Lablup 신정규 대표 EP.86 **"진짜 내 일을 해결하는 Agentic Workflow"** 강연을 기반으로,  
> 비개발자 PO가 Claude를 실무에 바로 활용할 수 있도록 재구성한 스터디 자료입니다.

---

## 🎯 이 커리큘럼을 마치면

- ✅ Claude와 단계적으로 대화하며 **PRD 초안을 40분 안에** 완성할 수 있다
- ✅ 나만의 **Soul Document**를 만들어 매번 같은 설명을 반복하지 않을 수 있다
- ✅ 스프린트 리뷰·기능 분석 등 반복 업무를 **명령어 하나**로 자동화할 수 있다
- ✅ 에이전트에게 검토를 맡겨 **내가 놓친 엣지케이스**를 발견할 수 있다
- ✅ 나만의 **PO-Claude 업무 Workflow**를 설계할 수 있다

## 🗺️ 내 상황에 맞는 학습 경로

### 🟢 완전 처음이라면 — 2주 코스

```
1주차  1강(마인드셋) → 2강(프롬프팅) → 3강(Soul Document)
2주차  4강(PRD) → 5강(컨텍스트) → 6강(자동화)
이후   7강(에이전트) → 8강(Workflow 완성)
```

### 🟡 Claude 써봤는데 제대로 활용 못 하는 느낌이라면 — 1주 코스

```
3강(Soul Document) → 4강(PRD) → 6강(자동화) → 8강(종합)
```

### 🔴 오늘 당장 써야 한다면 — 당일 코스

```
2강(프롬프팅 패턴) 빠르게 읽기
→ templates/ 에서 PRD 템플릿 가져가기
→ 부록 프롬프트 복사해서 사용
```

## 💬 이 커리큘럼의 핵심 철학

> *"내 걸 만들게 되면 그 가속이 시작됩니다."*  
> — 신정규 (Lablup 대표)

남이 만든 프롬프트를 복사하는 것도 좋지만,  
**나만의 시스템을 만드는 순간** 진짜 가속이 시작됩니다.

이 커리큘럼은 단순히 "Claude 잘 쓰는 법"이 아니라,  
**나만의 AI 협업 시스템을 만드는 과정**입니다.
